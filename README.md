# SP de Quebrada (PT-BR)

Pack de sons de um paulistano da quebrada, em portugues brasileiro, no formato [OpenPeon/CESP v1.0](https://openpeon.com/spec).

## Instalacao

```bash
peonping install sp_quebrada_pt-br
```

## Categorias

| Categoria | Sons | Descricao |
|---|---|---|
| `session.start` | 4 | Saudacao ao iniciar sessao |
| `task.acknowledge` | 4 | Confirmacao de tarefa |
| `task.complete` | 5 | Tarefa concluida |
| `task.error` | 5 | Reacao a erro |
| `input.required` | 3 | Esperando input do usuario |
| `resource.limit` | 3 | Recurso no limite |
